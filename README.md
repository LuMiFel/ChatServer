<p>-------------------------------------------</br>
Chat Commands:</br>
.help      (Show chat commands)</br>
.username  (Change your user name)</br>
.color     (Change your user color)</br>
.list      (Show list of users in chat)</br>
.mute      (Mute user messages)</br>
.quit      (Turn off connection)</br>
